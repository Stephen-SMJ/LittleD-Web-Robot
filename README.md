# LittleD Web Robot
## 小D是一个在低代码平台上的指引机器人，当用户进入平台时，可以通过小D快速创建应用和模板
使目前主要功能:
（一）页面与功能跳转：语音或文字输入跳转到XXX页面/系统/ 我要设计一个XXX系统，即可跳转到对应页面或模板。
（二）发送邮件：使用ASRT进行语音转换。多段交互式，1.发送给XXX 2.输入邮件主题 3.输入邮件内容。
（三）智能手绘表单生成：开发人员可以在纸上手绘表单，上传至系统自动生成相同的表单样式。
